1.  프로젝트 최상위 디렉터리 (StockPIPApp/)
    * `.idea/`
        * IntelliJ IDEA 통합 개발 환경(IDE)의 프로젝트 설정 파일들이 저장되는 숨김 폴더입니다. IDE가 프로젝트를 인식하고 관리하는 데 사용됩니다.
    * `out/`
        * 자바 소스 코드가 컴파일된 `.class` 파일 및 기타 빌드 결과물이 저장되는 폴더입니다.
    * `src/`
        * 모든 자바 소스 코드와 리소스 파일들이 위치하는 루트 폴더입니다.
        * `main/`
            * 주요 애플리케이션 코드를 포함하는 폴더입니다.
            * `java/`
                * 자바 소스 코드(`.java` 파일)가 저장되는 핵심 폴더입니다. 이 폴더는 IntelliJ IDEA에서 '소스 루트'로 지정됩니다.
                * `java3team/PIPApp/`
                    * 애플리케이션의 모든 자바 패키지가 시작되는 최상위 패키지입니다. 팀의 고유한 도메인 이름을 역순으로 사용하여 이름 충돌을 방지하고 코드 모듈화를 돕습니다.
                    * `ui.Main.java`
                        * 애플리케이션의 주 진입점(main 메서드)을 포함하는 클래스입니다. JavaFX 애플리케이션의 시작을 담당합니다.
                    * `ui/`
                        * 사용자 인터페이스(UI) 관련 클래스들을 모아두는 패키지입니다.
                        * `controller/`
                            * UI 요소와 사용자 상호작용을 처리하는 컨트롤러 클래스들을 포함합니다.
                        * `util/`
                            * UI 관련 유틸리티 기능 또는 헬퍼 클래스들을 포함합니다.
                        * `StockView.java`
                            * 주식 정보 표시 화면과 관련된 UI 로직을 담당하는 클래스입니다.
                        * `SettingsWindow.java`
                            * 애플리케이션 설정 창과 관련된 UI 로직을 담당하는 클래스입니다.
                    * `api/`
                        * 외부 API (예: Finnhub API) 연동 및 데이터 모델을 정의하는 패키지입니다.
                        * `finnhub/`
                            * Finnhub API와 관련된 구체적인 구현을 포함하는 패키지입니다.
                            * `client/`
                                * 실제로 Finnhub API에 HTTP 요청을 보내고 응답을 처리하는 클라이언트 로직을 포함합니다.
                            * `model/`
                                * API 응답으로 받는 JSON 데이터를 자바 객체로 매핑하기 위한 데이터 모델 클래스들(예: `StockQuote`, `CompanyProfile`)을 포함합니다.
                        * `FinnhubApiClient.java`
                            * Finnhub API와의 통신을 전담하는 클래스입니다.
                    * `service/`
                        * 애플리케이션의 핵심 비즈니스 로직 및 데이터 처리 기능을 구현하는 서비스 클래스들을 모아두는 패키지입니다.
                        * `StockService.java`
                            * 주식 데이터 조회 및 처리와 관련된 비즈니스 로직을 담당하는 클래스입니다.
                        * `PreferencesManager.java`
                            * 애플리케이션 설정 저장 및 로드와 같은 사용자 환경설정 관리를 담당하는 클래스입니다.

                    * 'config/'
                       * App Constants: 전역변수를 다루는 클래스파일
    * `.gitignore`
        * 버전 관리 시스템(Git)이 추적하지 않아야 할 파일이나 폴더(예: IDE 설정 파일, 컴파일된 파일) 목록을 정의하는 파일입니다.
    * `README.md`
        * 프로젝트에 대한 설명, 설정 방법, 실행 방법 등 전반적인 정보를 담는 문서 파일입니다.
    * `StockPIPApp.iml`
        * IntelliJ IDEA가 이 프로젝트를 모듈로 인식하는 데 필요한 설정 정보를 담고 있는 파일입니다. (XML 형식)
